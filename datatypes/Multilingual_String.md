# Multilinqual String

One or several [Strings](../datatypes/String.md) qualified by distinct [Language Tags](../datatypes/Language_Tag.md).

---
## Matches

This datatype matches the `rdf:PlainLiteral`<sup>[1](#fn1)</sup>.
